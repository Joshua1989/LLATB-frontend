Edit Your User Profile

Using Third-Party Proxy

If you are using a third-party proxy SIFStats (JP), LLproxy (CN), then you can directly copy the user profile JSON to input your card list.

Manually Edit Card Profile

If you are not use a third-party proxy, you can still use this tool by editing your profile manually. The editing is incremental, if you first copy user profile JSON and then edit manually, you will start from the profile in the JSON.

Or if you have the data from minaraishi's team builder, you can also copy it here, now LLATB supports minaraishi's format!

When you add a card, first you need to choose which member, attribute and whether it is idolized to filter out all card satisfying these conditions.

Then two sets of card will show below:

  • the first set contains all possible cards available in the game that satisfies the filter
  • the second set contains all your cards that satisfies the filter

You can add a new card by clicking an icon in the first set, then it will appear in the second set, by default the card has minimum number of slots, skill level 1, max level and bond.

You can edit an existing card by clicking an icon in the second set, then the following modal will pop up for you to change card stats. The only thing you need to notice is that if you changed the idolization, the edited will disappear from the second set since it does not satisfy the current filter (idolization does not match).

After you finished editing all your cards, you may want to edit the number of all gems, simply click "Edit Gem" button and input the numbers.

Now you are done, it is simple right? Do not forget to click "Save" before you click "Build" to build your optimal team. And I highly recommend you to save the user JSON locally in case of losing information.

Build Optimal Team

Configure Settings

Once you created a valid user profile, simply choose all parameters:

  • Live List: the live(s) you want to optimize over: this team builder support Medley Festival, under same group, attribute and difficulty, you can add at most 5 songs
  • Buy Support: this is useful when you are in a Medley Festival or Challenge Festival
  • Prior Perfect Rate: this is used for evaluating how well you play the game, it is the average probability that you tap a PERFECT in absence of any perfect lock skills. Note that all computation are based on Full Combo presumption, Prior Perfect Rate is 0.95 means your taps are 95% PERFECT and 5% GREAT.
  • Gem Unlimited: just a setting to discard gem owning constraints, this may tell you what gem is more valuable in your following game progress
  • Extra Condition: you can choose extra conditions to see your potentials:
    • Maintain Current: nothing changes, cards information strictly follows what you input in user profile JSON
    • Max Lv & Bond: All cards maintain current idolization, but are set to max level and max bond
    • Idlz, Max Lv & Bond: All cards set to idolized, but are set to max level and max bond, slot number unchanged
    • Idlz, Lv 8 & Slot Max: This is the ultimate status, all cards set to idolized, but are set to max level and max bond, besides the skill level is 8 and maximize the slot number.
